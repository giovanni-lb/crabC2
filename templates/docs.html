Infected Machine

All infected machines could be found in the machine dashboard

View machine information

When a new machine is infected, our C2 retrieve automaticly some information about the system (user, mac address, IP, hostname), you can see all thoses information by clicking the view button.

Machine dashboard

View machine content

How to add an infected machine

All machines infected using the 1st stage generator (generate your 1st stage) will be register as a new infected machine. You can manually add one using the /get-info endpoint , but it's not the right way to do it
